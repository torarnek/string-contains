# string-contains

> Function to check if a string contains another string

## Install
Install with [npm](http://www.npmjs.org):

```bash
npm install string-contains --save-dev
```

## Usage

```js
var contains = require("string-contains");
console.log(contains("Hello world", "world"));
//=> 'true'
```
